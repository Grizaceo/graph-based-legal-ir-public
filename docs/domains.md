# Strategic Domains

The project is deliberately domain-bounded. We are not trying to map all law at once; we are proving the method in a few high-value domains first.

---

## Current focus

| Domain | Role | Why it matters |
|---|---|---|
| Labor law | Primary pilot | High practical value and a strong fit for evidence-first retrieval |
| Environmental law | Secondary pilot | Good testbed for administrative and judicial grounding |
| Procedural support | Application layer | Useful for deadline and workflow assistance, but not a core knowledge domain |

---

## How a new domain is added

1. identify approved source families
2. sample canonical material for review
3. define the domain-specific evidence rules
4. update prompting and retrieval heuristics where necessary
5. run a review sprint before any canonical promotion

---

## Expansion discipline

A new domain should not be admitted just because it is interesting.
It should be admitted when:
- the sources are stable enough
- the evidence model still works
- the review path remains human-governed
- the public story stays honest about what is actually supported

## Domain principle

The system scales by repeating the method, not by relaxing the standard.
