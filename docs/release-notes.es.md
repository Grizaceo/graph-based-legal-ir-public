# Notas de Versión

## Qué hace esta versión

Esta versión convierte el repositorio en un harness público limpio para el ecosistema de IR legal LexCon / LexO.
Está pensada para ser leída por humanos primero: colaboradores potenciales, revisores y cualquiera que intente entender el sistema sin necesidad del stack privado.

## Por qué importa

El IR legal a menudo se presenta como si el ranking fuera suficiente.
Esta versión dice lo contrario: en el trabajo legal, un sistema útil debe mostrar su evidencia, reconocer sus límites y negarse a exagerar su confianza.

## Qué cambió en esta versión pública

- nombres y referencias del ecosistema actualizados para reflejar el stack actual LexCon / LexO
- referencias obsoletas eliminadas que hacían ver la historia como un prototipo interno
- narrativa de lanzamiento dedicada para visitantes de GitHub
- límite entre documentación pública y operaciones privadas ajustado
- hoja de ruta alineada con objetivos de publicación pública basada en evidencia

## Resumen de cambios

- arquitectura reescrita alrededor del stack actual de IR legal en capas
- página del ecosistema actualizada para mostrar el mapa de componentes actual
- estado actual acotado a afirmaciones honestas y públicas
- hoja de ruta reenfocada como una ruta de publicación pública por etapas
- plan de demo reescrito para mostrar comportamiento acotado y fail-closed
- política de repositorio clarificada para higiene de publicación pública

## Frase pública

Una versión de documentación pública para IR legal chileno explicable y basado en evidencia.
